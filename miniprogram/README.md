# 农智通微信小程序

农智通农技服务一体化平台的微信小程序端，提供用户端的所有功能。

## 项目结构

```
miniprogram/
├── app.json          # 小程序配置文件
├── app.js            # 小程序入口文件
├── app.wxss          # 全局样式文件
├── pages/            # 页面目录
│   ├── index/        # 首页
│   ├── category/     # 分类页
│   ├── cart/         # 购物车页
│   ├── profile/      # 个人中心页
│   ├── login/        # 登录页
│   ├── productDetail/ # 产品详情页
│   ├── order/        # 订单页
│   └── orderDetail/  # 订单详情页
└── icons/            # 图标目录
```

## 功能模块

1. **首页**：展示产品列表、轮播图、导航标签
2. **分类**：展示产品分类和分类下的产品
3. **购物车**：管理购物车商品、调整数量、结算
4. **个人中心**：用户信息、订单管理、功能入口
5. **登录**：用户登录功能
6. **产品详情**：展示产品详细信息、加入购物车、立即购买
7. **订单**：展示订单列表、按状态筛选
8. **订单详情**：展示订单详细信息、取消订单、支付订单

## 开发环境

1. **微信开发者工具**：用于开发和调试小程序
2. **Node.js**：用于运行后端服务
3. **微信小程序账号**：用于发布小程序

## 如何使用

1. 下载并安装微信开发者工具
2. 打开微信开发者工具，点击「导入项目」
3. 选择本项目的 `miniprogram` 目录
4. 填写小程序的 AppID（如果没有，可以使用测试号）
5. 点击「导入」按钮
6. 等待项目加载完成，即可在开发者工具中预览和调试

## 后端服务

小程序需要与后端服务配合使用，后端服务地址配置在 `app.js` 文件中：

```javascript
this.globalData.apiBaseUrl = 'http://localhost:3000/api';
```

请确保后端服务已经启动并运行在指定地址。

## 注意事项

1. 本项目使用了模拟数据，实际项目中需要替换为真实的API调用
2. 小程序需要在微信公众平台中配置合法的域名，否则无法调用后端API
3. 请确保后端服务已经启动，否则小程序无法正常获取数据
