# 农智通农技服务一体化平台

## 项目简介

农智通是一个集农资产品销售、农技服务于一体的综合性平台。为农户提供一站式的农业生产解决方案。

**核心特点：免费、本地运行、无需外网连接**

## 快速开始

### 方式一：双击启动（推荐）

1. 进入项目目录：`d:\三创\nongzhitong`
2. 双击运行 `start.bat` 或 `start-server.ps1`
3. 浏览器自动打开 http://localhost:3000/user/index.html

### 方式二：命令行启动

```bash
cd d:\三创\nongzhitong\backend
npm install   # 首次运行需要安装依赖
npm start     # 启动服务
```

## 技术架构

### 前端
- HTML5 + CSS3 + JavaScript（原生）
- 响应式设计，支持手机和电脑
- 无需任何框架，浏览器即可运行

### 后端
- Node.js + Express
- SQLite数据库（内置，无需安装）
- JWT认证

### 数据库
- SQLite：轻量级数据库，随项目自动创建
- 无需安装MySQL等外部数据库软件

## 常见问题

### Q: 如何让手机访问？
确保手机和电脑在同一WiFi网络，然后访问 `http://电脑IP:3000`

### Q: 端口被占用怎么办？
```bash
set PORT=3001 && npm start
```

### Q: 如何查看验证码？
验证码会在启动服务的命令行窗口输出。

### Q: 支付需要真实付费吗？
不需要，当前是模拟支付，适合演示和测试。
